# 🎮 Akinator – Devineur de Pays (C / Linux)

## 📌 Description
Application développée en C sous Linux qui devine le pays européen auquel pense l’utilisateur grâce à un arbre de décision dynamique.

Interface graphique réalisée avec GfxLib.

## ⚙️ Fonctionnalités
- Arbre de décision fonctionnel
- Interface graphique interactive (Oui / Non)
- Sauvegarde automatique des nouvelles réponses
- Possibilité de rejouer sans relancer le programme

## 🛠️ Stack technique
- C
- Linux (Ubuntu VM)
- GfxLib
- Makefile
- Gestion de fichiers

## 🚀 Lancer le projet
```bash
make
./pays
